### COMO UTILIZAR O PROJETO ###

1.ATENÇÃO: É NECESSÁRIO QUE TENHA SIDO INSTALADO O NODE EM SUA MÁQUINA ANTES DOS PRÓXIMOS PASSOS

2.Abra o cmd 

3.navegue até o diretório da página do projeto utilizando: cd(para entrar no diretório) "[diretório da pasta do projeto]"
EXEMPLO: cd "C:\...\Github\PROJETO-RAMO" e aperte enter

4.Use o comando npm install (este comando irá instalar as dependências que estão no package.json)

5.Após instalar tudo, o projeto já dá para ser utilizado na sua máquina

6.Para rodar o projeto em um servidor: digite no diretório do projeto - node index.js ou nodemon index.js 

7.Vá no navegador e digite: http://localhost:3001
